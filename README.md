## Présentation

AfriTalent est une plateforme web qui connecte les freelances africains aux entreprises à la recherche de talents qualifiés dans différents domaines tels que le développement web, le design, le marketing digital, la cybersécurité et la data science.

## Fonctionnalités

* Page d'accueil moderne et responsive
* Catalogue de freelances
* Présentation des tarifs
* Blog d'actualités
* Formulaire de contact avec validation JavaScript
* Mode sombre
* Animations et effets visuels
* Compatible mobile, tablette et ordinateur

## Technologies utilisées

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* Google Fonts
